# Exercise demo photos

Self-hosted copies of the demo photos from
[free-exercise-db](https://github.com/yuhonas/free-exercise-db), an open
public-domain exercise dataset released under
[The Unlicense](https://unlicense.org/). Served via this repo's GitHub Pages
site and referenced by the `exercises.images` URLs seeded in
`backend/app/seed/exercise_library.py`.
